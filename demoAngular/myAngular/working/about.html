About My ASP.NET design


This demo to demonstrate my understanding of AngularJS. AngularJS is a structural framework for dynamic web applications and can be used in many systems. My designs are all based on Angular, bootstrap3 and jQuery. JQuery is only used in about page to handle the images and some others. The data shown on the list page which is defined in Angular controllers methods and the page layout is referenced by bootstrap3. All the data can be replaced when the system connects to a real Database. The source code editing and testing are in Visual Studio community 2015.
